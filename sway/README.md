# Keybinds

| Key bind          | The actual key  | Function                                                 |
| ----------------- | --------------- | -------------------------------------------------------- |
| `$mod+w`          | `Windows+w`     | Excutes `wall-switch.sh` wallpapwer changer script       |
| `$mod+Return`     | `Windows+Enter` | Opens $term # Start a primary terminal (**Kitty**)       |
| `$mod-alt+Return` | `Alt+Enter`     | Opens `~/.config/st/st` (**suckless terminal**)          |
| `$mod+q`          | `Windows+q`     | Kill/quit window                                         |
| `$mod-alt+Ctrl+l` | `Alt+Ctrl+l`    | Screen Lock (**swaylock**) script                        |
| `$mod+d`          | `Windows+d`     | Excutes `launcher.sh` rofi application launcher script   |
| `$mod+m`          | `Windows+m`     | Launch **Nautilus** file manager                         |
| `$mod-alt+a`      | `Alt+a`         | Excutes `controllers.sh` script                          |
| `F6`              | `F6`            | Excutes **brightnessctl** to decrease brightness by `5%` |
| `F7`              | `F7`            | Excutes **brightnessctl** to increase brightness by `5%` |
| `F1`              | `F1`            | Excutes **pactl** to move volume to 0%                   |
| `F2`              | `F2`            | Excutes **pactl** to decrease volume by 5%               |
| `F3`              | `F3`            | Excutes **pactl** to increase volume by 5%               |
| `$mod-alt+n`      | `Alt+n`         | Excutes `wifi-menu.sh` script to launch wifi menu        |

and many more...
